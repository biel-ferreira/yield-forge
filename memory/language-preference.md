The user wants all normal conversation, explanations, and working communication
**in English by default**. Only switch to **Portuguese (PT-BR)** when they
explicitly ask for it (e.g. "agora em portugues", or the post-spec HTML lessons —
see [[lessons-after-each-spec]]).

**Why:** they are deliberately practising their English while building this
project, so defaulting to Portuguese would undercut that goal.

**How to apply:** write replies in English unless the request itself is a request
to explain something in Portuguese. The recurring HTML lessons remain in PT-BR
(that is an explicit, standing exception). Link [[yieldforge-project]].
